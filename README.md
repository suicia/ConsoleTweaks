# ConsoleTweaks
## Docs
- **Write()**
**you can use Write() to output data in the console**
```c#
  Tweak.Write("Hello green world!");
```
**you can also assign a color to each character or passage**
```c#
  Tweak.Write("~r~Hello ~g~green ~db~world!");
```
